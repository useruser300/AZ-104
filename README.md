# AZ-104T00-A: Microsoft Azure Administrator

This repository contains my notes and study materials for the **AZ-104T00-A: Microsoft Azure Administrator** course.

The course focuses on the main responsibilities of an Azure Administrator, including managing identities, governance, storage, compute resources, virtual networks, monitoring, backup, and disaster recovery in Microsoft Azure.

## Course Overview

The AZ-104 course teaches how to implement, manage, and monitor Azure resources in a cloud environment.

It covers important administration tasks such as:

- Managing Azure subscriptions and resources
- Securing identities with Microsoft Entra ID
- Configuring governance and role-based access control
- Managing storage solutions
- Creating and managing virtual machines
- Configuring virtual networking
- Connecting Azure with on-premises environments
- Managing network traffic
- Implementing backup and recovery
- Monitoring Azure resources

## Repository Purpose

The purpose of this repository is to organize my AZ-104 learning notes in a clear and structured way.
